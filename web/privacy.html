适用范围

本隐私政策适用于 Classmate 课程表管理软件及其与之直接相关的本地功能、网页解析导入功能和公示页面。本政策旨在说明在你使用本应用过程中，应用会在何种情况下处理你的信息、处理何种类型的信息、将信息保存于何处，以及在何种情况下可能与外部系统发生交互。

需要说明的是，本应用为本地优先的软件产品，不以建立用户账户体系、持续上传用户数据或实施行为画像为基础运行模式。因此，本政策所述的信息处理方式主要围绕本地存储、文件导入导出、网页内容解析、系统分享以及外部链接跳转等场景展开。

本地存储

课表数据和相关设置会保存在应用文档目录中的本地文件 classmate_data.json，可编辑的学校站点配置会单独保存在 classmate_school_sites.json。自定义课表解析设置中填写的 Base URL、API 密钥和所选模型也会随同应用数据保存在本地，当前实现不会使用系统安全凭据库进行保护。若在浏览器中使用，同类数据会保存在浏览器本地存储中。应用不会自动把这些本地数据上传到开发者控制的服务器。

除你主动触发特定外部交互功能外，本应用不会自动将上述本地数据上传至开发者控制的服务器，也不会在后台持续传输你的课表内容、设置内容或学校站点配置内容。

导入与导出

当且仅当你主动选择导入文件、主动保存文件或主动执行导出操作时，应用才会访问你指定的本地文件内容或向你指定的位置写出文件。该类文件可能包括 JSON 课表文件、学校站点配置文件及节次模板文件。

文件导入导出本身属于本地设备上的数据操作。除非你进一步主动选择网页解析导入，否则单纯的导入导出行为不会将你的文件内容自动发送至外部解析接口或第三方服务。

分享功能

当你主动使用分享功能时，应用会将你明确选定的导出文件交由系统分享面板或你选择的目标应用处理。该操作属于你主动发起的设备级分享行为。分享后的文件如何被保存、转发或进一步处理，将取决于你所选择的目标应用、目标服务或接收方的处理规则。

本应用无法控制第三方应用在接收文件后的处理方式，也无法替代第三方对文件进行保密、删除或限制传播。为此，请你在使用分享功能前自行评估文件内容及分享对象。

外部链接

当你主动打开 GitHub 仓库、官方网站或其他外部链接时，应用会调用系统浏览器或其他外部应用完成访问请求。此类访问行为一旦离开本应用控制范围，相关数据处理即受对应第三方网站、平台或应用自身的规则和政策约束。

因此，请你在访问外部链接前自行审阅第三方平台的相关政策，并对其信息处理方式作出独立判断。

默认不收集的内容

在默认状态下，本应用不要求你注册 Classmate 账号，不默认启用分析统计、个性化广告、广告标识符跟踪、云端画像或云端自动备份功能。本应用本身也未提供专门用于采集学校账号密码的独立输入字段。

若你在应用内打开的学校网页中输入账号、密码或其他学校系统凭据，该交互发生在你所访问的学校网页内，而非由本应用定义的账号体系之中。本应用仅提供网页容器或网页内容导入能力，并不以自身名义存储、管理或验证学校凭据。

本应用不包含崩溃报告、错误日志自动上报或使用情况遥测功能。应用启动时会在后台静默请求隐私政策公示页面以检测版本更新，该请求不会携带你的课表数据或个人标识信息。

网页解析与接口转发

当你使用学校网页导入或粘贴 HTML 进行解析时，应用会先在本地压缩内容，再把你提交的页面内容、可选的页面标题与 URL、当前应用语言以及解析提示词发送到当前选择的解析接口。如果你使用官方解析，请求会发往应用配置的官方后端；如果你启用自定义 OpenAI 兼容解析，同样的内容会直接发送到你填写的第三方接口，获取模型列表时也会请求该第三方接口。

你应当了解，解析接口可能由其部署方进一步配置，将请求转发至 AI 服务或其他上游处理服务。当前官方后端会限制单次提交内容不得大于 300KB，使用其配置的请求超时时间，并按 IP 每天最多解析 5 次。解析接口及上游服务如何处理数据，取决于相关部署方、服务提供方或模型提供方的处理规则。

当前项目默认后端逻辑中，网页解析请求存在如下技术性限制：单次提交内容不得大于 300KB；接口按照配置的超时时间执行请求；并按 IP 每天最多解析 5 次。上述限制可能随部署方式调整而变化。

用户注意事项

  • 你应确保导入、导出、分享或解析的内容系你有权处理的信息；
  • 你应在使用学校网页解析前，自行判断相关页面内容是否包含不宜提交至解析接口的信息；
  • 你应妥善保管导出的文件，避免在不受信任的环境中传播；
  • 如你通过外部网站、浏览器或第三方应用完成后续操作，应自行承担对第三方规则进行审查与选择的责任。

本应用提供的是课程表管理与辅助导入工具，不对第三方平台的稳定性、持续可用性、内容准确性或数据处理方式作出额外保证。

政策更新

若后续版本对信息处理方式、功能边界、第三方接口使用情况或其他重要事项进行了调整，本页面内容将同步更新。必要时，应用可能要求你重新阅读并确认更新后的隐私政策后，方可继续使用部分功能。
